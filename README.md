# mtm6201-final

Overview:
This project is a personal portfolio website designed to showcase my work in 3D rendering, UX/UI design, and graphic design. The goal was to create a visually engaging and user-friendly site that reflects my design style and technical skill

Responsive Layout Issues:
Challenge: Getting elements to align properly across different screen sizes.
Solution: I used Bootstrap’s grid system (col-, row, container) and tested breakpoints to ensure consistency on mobile, tablet, and desktop.

Key Lessons & Subjects Learned:
How to build fully responsive layouts using Bootstrap (Desktop, Tablet, Mobile)
  - Responsive through bootstrap elements such as: py, col
How to use CSS transitions and transforms for interactive UI
  - Mades website more engaging and unique!
The importance of clean HTML structure
How to debug layout and functionality issues efficiently
How to translate design mockups into real, functional code

Technologies Used:
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
